# 学生互评管理系统

> 本项目为学生互评管理系统的前端部分。

### 技术选型

脚手架：create-react-app

1. 技术栈：react

2. 组件库：ant design

3. 路由：react-router-dom

4. 数据流：redux + redux-thunk + redux-persist

5. CSS预处理：sass

---

### 开发记录

1. 对于绑定数据较少的组件，直接使用react-redux hooks进行数据绑定和dispatch，提高性能。对于依赖数据较多的组件（比如整个页面），考虑使用connect。
